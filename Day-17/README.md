## ✅ Problem Solved

### 1. Merge Intervals (56)
- Approach: Sorting + Greedy  
- Time Complexity: O(n log n)  
- Space Complexity: O(n)  

---

## 💡 Approach Explanation

- Sort intervals based on starting time  
- Initialize result list with first interval  
- Compare current interval with last merged interval  
- If overlapping → merge them  
- Else → add new interval  

---

## 📚 Key Learnings

- Importance of sorting in interval problems  
- Greedy approach for merging  
- Handling overlapping ranges efficiently
