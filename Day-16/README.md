# 📅 Day 16 — #DrGViswanathanChallenge

##  Problem Solved

### 1. Rank Transform of an Array (1331)
- Approach: Sorting + HashMap  
- Time Complexity: O(n log n)  
- Space Complexity: O(n)  

---

## Approach Explanation

- Copy and sort the array  
- Assign ranks to unique elements  
- Store ranks using HashMap  
- Replace original elements with their rank  

---

##  Key Learnings

- Combining sorting with hashing  
- Efficient mapping techniques  
- Handling duplicates properly  

---

##  Notes

A great problem to understand ranking logic and mapping values efficiently.
