## ✅ Problem Solved

### 1. Valid Parentheses (20)
- Approach: Stack  
- Time Complexity: O(n)  
- Space Complexity: O(n)  

---

## 💡 Approach Explanation

- Traverse the string  
- Push opening brackets onto stack  
- For closing brackets → check top of stack  
- If mismatch or empty → return false  
- At the end, stack should be empty  

---

## 📚 Key Learnings

- Stack for matching problems  
- Handling brackets efficiently  
- Importance of LIFO structure
