## ✅ Problem Solved

### 1. Find First and Last Position of Element in Sorted Array (34)
- Approach: Binary Search (Lower Bound + Upper Bound)  
- Time Complexity: O(log n)  
- Space Complexity: O(1)  

---

## 💡 Approach Explanation

- Use Binary Search twice:
  1. First → find first occurrence  
  2. Second → find last occurrence  

- First Occurrence:
  - Move left when target found  

- Last Occurrence:
  - Move right when target found  

---

## 📚 Key Learnings

- Advanced Binary Search  
- Boundary searching  
- Efficient searching in sorted arrays
