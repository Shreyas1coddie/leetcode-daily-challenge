# 📅 Day 13 — #DrGViswanathanChallenge

## ✅ Problem Solved

### 1. Ugly Number (263)
- Approach: Iterative Division (2, 3, 5)
- Time Complexity: O(log n)
- Space Complexity: O(1)

---

## 💡 Approach Explanation

- If n ≤ 0 → return false  
- Keep dividing the number by 2, 3, and 5 if divisible  
- Repeat until n becomes 1  
- If at any step number is not divisible → return false  

---

## 📚 Key Learnings

- Breaking problems using loops  
- Number reduction using division  
- Handling edge cases effectively
