# 📅 Day 14 — #DrGViswanathanChallenge

## ✅ Problem Solved

### 1. Intersection of Two Linked Lists (160)
- Approach: Two Pointer Technique  
- Time Complexity: O(n + m)  
- Space Complexity: O(1)  

---

## 💡 Approach Explanation

- Use two pointers for both lists  
- Traverse both lists  
- When one pointer reaches null, redirect it to the other list  
- Eventually, both pointers meet at intersection node or null  

---

## 📚 Key Learnings

- Handling unequal linked list lengths  
- Smart pointer switching technique  
- Avoiding extra space and length calculation  

---

A very elegant problem that demonstrates how logic can replace brute-force approaches.
