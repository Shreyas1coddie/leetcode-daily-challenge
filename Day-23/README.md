# 📅 Day 23 — #DrGViswanathanChallenge

## ✅ Problems Solved

### 1. Rotate Image (48)
- Approach: Transpose + Reverse  
- Time Complexity: O(n²)  
- Space Complexity: O(1)  

### 2. Palindrome Number (9)
- Approach: Reverse Number  
- Time Complexity: O(log n)  
- Space Complexity: O(1)  

---

## 💡 Approach Explanation

### Rotate Image
- Transpose the matrix (swap rows & columns)  
- Reverse each row  
- Result: 90° clockwise rotation  

### Palindrome Number
- Reverse the digits of the number  
- Compare with original number  
- If equal → palindrome
