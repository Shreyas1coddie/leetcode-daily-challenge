# 📅 Day 04 — #DrGViswanathanChallenge

## ✅ Problem Solved

### 1. Container With Most Water (11)

* Approach: Two Pointer Technique
* Time Complexity: O(n)
* Space Complexity: O(1)

---

## 📚 Key Learnings

* Two pointers can reduce time complexity from O(n²) to O(n)
* Always move the pointer with smaller height
* Greedy thinking helps optimize brute force solutions

---

## 🚀 Notes

Initially approached the problem with brute force, but optimized it using the two-pointer technique which made it much more efficient.
