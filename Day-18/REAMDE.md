# 📅 Day 18 — #DrGViswanathanChallenge

## ✅ Problem Solved

### 1. Next Greater Element
- Approach: Stack (Monotonic Stack)  
- Time Complexity: O(n)  
- Space Complexity: O(n)  

---

## 💡 Approach Explanation

- Traverse the array  
- Use a stack to store elements  
- For each element, pop smaller elements from stack  
- The next greater element is found when a larger element appears  
- Push current element to stack  

---

## 📚 Key Learnings

- Monotonic Stack concept  
- Efficient traversal for next greater problems  
- Stack-based optimization  

---

## 🚀 Notes

A very important pattern used in many interview problems like stock span, daily temperatures, etc.
