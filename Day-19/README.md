# 📅 Day 19 — #DrGViswanathanChallenge

## ✅ Problem Solved

### 1. Search Insert Position (35)
- Approach: Binary Search  
- Time Complexity: O(log n)  
- Space Complexity: O(1)  

---

## 💡 Approach Explanation

- Initialize low = 0, high = n-1  
- Use binary search to find target  
- If target found → return index  
- If not found → return position where it should be inserted  
- Maintain answer variable to track insert position  

---

## 📚 Key Learnings

- Binary Search fundamentals  
- Handling edge cases (not found case)  
- Efficient searching in sorted arrays  

---
