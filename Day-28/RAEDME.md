### 1. Preorder Traversal (144)
- Order: Node → Left → Right  

### 2. Postorder Traversal (145)
- Order: Left → Right → Node  

---

## 💡 Approach

- Used **Recursive DFS**
- Base case: if node is null → return
- Follow traversal order while adding values to result list  

---

## 📚 Key Learnings

- Preorder is useful for:
  - Copying trees
  - Tree construction  

- Postorder is useful for:
  - Deleting trees
  - Bottom-up calculations
