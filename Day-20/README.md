# 📅 Day 20 — #DrGViswanathanChallenge

## ✅ Problem Solved

### 1. Generate Parentheses (22)
- Approach: Backtracking (Recursion)  
- Time Complexity: O(2ⁿ)  
- Space Complexity: O(n)  

---

## 💡 Approach Explanation

- Use recursion to build the string step by step  
- Add '(' when open < n  
- Add ')' when close < open  
- Ensure only valid combinations are generated  
- Base case: when string length = 2 * n → add to result  

---

## 📚 Key Learnings

- Backtracking technique  
- Handling constraints in recursion  
- Generating all valid combinations  

---

## 🚀 Notes

A very important problem to understand recursion and backtracking.  
Helps build strong foundation for solving combination-based problems.
