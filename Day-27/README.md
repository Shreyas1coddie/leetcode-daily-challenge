## ✅ Problem Solved

### 1. Binary Tree Inorder Traversal (94)
- Approach: Recursion (DFS)
- Time Complexity: O(n)
- Space Complexity: O(n)

---

## 💡 Approach Explanation

- Traverse the left subtree  
- Visit the root node  
- Traverse the right subtree  

This follows the **Inorder (L → R → R)** pattern.

---

## 📚 Key Learnings

- Depth First Search (DFS)  
- Recursive tree traversal  
- Understanding traversal orders  

---

## 🚀 Notes

Inorder traversal is fundamental and often used in BST problems.
